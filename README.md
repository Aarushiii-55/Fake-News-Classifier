# Fake-News-Classifier


A machine learning pipeline to detect and classify fake news articles with **92–94% accuracy** across 100,000+ records using ensemble models and NLP techniques.

---

## 📊 Results

| Metric | Value |
|---|---|
| Accuracy | **92–94%** |
| Dataset Size | 100,000+ records |
| Technique | Ensemble Methods + Feature Engineering |

---

## 🧠 Approach

- **Text Preprocessing** — Tokenization, stopword removal, TF-IDF vectorization
- **Feature Engineering** — Linguistic features, metadata extraction
- **Ensemble Modeling** — Combined multiple classifiers for robust predictions
- **Hyperparameter Tuning** — Optimized for best accuracy and generalization
- **Cross-Validation** — Ensured model robustness across data splits

---

## ⚙️ Tech Stack

- **Language:** Python
- **NLP:** NLTK, TF-IDF, Scikit-learn
- **Models:** Random Forest, XGBoost, Ensemble Methods
- **Data Processing:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn
